# NHIỆM VỤ CHÍNH
Đây là tài liệu hướng dẫn sử dụng và phát triển dự án Memory Graph. Mục tiêu chính của dự án là xây dựng một hệ thống quản lý bộ nhớ hiệu quả, hỗ trợ lưu trữ và truy xuất thông tin một cách nhanh chóng và chính xác.

- Chống AI quên lãng thông tin quan trọng bằng cách sử dụng đồ thị tri thức để tổ chức dữ liệu.

**VAI TRÒ**:
- Audit
- Timeline
- Chống ảo giác (fact anchoring)

# Flow hoạt động chuẩn của hệ thống quản lý bộ nhớ:
```
Goal
↓
Decision
↓
Action
↓
Observation
↓
Error? → Fix → Lesson
↓
Graph update
↓
Memory embedding
```
